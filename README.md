# javaTestExample

Za izvajanje programa:

1. Poženi program tako da dvakrat klikneš na jar file in izbereš verzijo OpenJDK Java 21 ali novejšo, ki ga uporabiš za izvajanje programa. Starejše verzije niso potrjene. Testirano z 21 in 26.

2. Ali poženeš v BASH s komando "java -jar pathTo/bankaTest-1.0.jar". Zamenjaj "pathTo" z lokacijo extrahiranega programa.
